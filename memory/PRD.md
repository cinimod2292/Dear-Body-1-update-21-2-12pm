# Dear Body - Beauty & Cosmetics Ecommerce Prototype

## Overview
Complete frontend UI prototype for Dear Body beauty ecommerce store redesign. Phase 1 deliverable with fully clickable storefront, shopping flow, and admin dashboard.

## Pages Built

### Storefront
- **Homepage** - Hero, categories, best sellers, new arrivals, testimonials, CTA
- **Product Listing** - Filterable grid with category sidebar, sorting, grid toggle
- **Product Detail** - Image gallery, pricing, descriptions, tabs, related products
- **Category Pages** - Dynamic filtered views per category
- **Search Results** - Full-text search across products
- **Wishlist** - Saved items with empty state

### Shopping Flow
- **Cart Page** - Full cart management, quantity controls, order summary
- **Cart Drawer** - Slide-out cart sidebar on add-to-cart
- **Checkout** - 3-step flow: Shipping → Payment → Review → Place Order
- **Order Confirmation** - Success page with order details

### Admin Dashboard
- **Dashboard** - Revenue chart, stats, recent orders
- **Orders** - Searchable/filterable order list with detail views
- **Order Detail** - Items, status update, shipping/payment info
- **Products** - Product management table with add/edit forms
- **Product Edit** - Full form with variants, images, tags
- **Customers** - Customer list with profile views
- **Refunds** - Refund management with status updates
- **Shipping** - Rate configuration and settings
- **Reports** - Revenue/orders/category charts with top products

## Tech Stack
- React 19, React Router, Tailwind CSS, shadcn/ui
- Framer Motion animations, Recharts
- Cormorant Garamond + Inter typography
- HSL-based design tokens with warm rose/blush palette

## Status
Phase 1 complete - All UI is MOCKED with local state (no backend). Cart/checkout/admin all simulate real flows.
